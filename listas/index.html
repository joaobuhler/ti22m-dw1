o elemento CODE serve para demonstrar um fragmento de código presente no texto

o elemento PRE serve para definir os espaços desse trecho de código

o elemento SAMP serve para mostrar a saída desse código em um fragmento de texto

o elemento KBD serve para mostrar um comando no teclado



Meus comandos favoritos do VScode são:

Ctrl + Shift + P
Que mostra uma lista de comandos do programa

Alt + Z
Que realiza a quebra de texto

Ctrl + Z
Que retorna as ações feitas pelo usuário, corrijindo possíveis erros

Ctrl + Space
Que mostra as "dicas" do VScode
